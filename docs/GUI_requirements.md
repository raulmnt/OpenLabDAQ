Tube Furnace DAQ GUI Requirements

Main Window

- Dynamically creates one plot for each enabled sensor.
- Sensor names come from config.json.
- Units come from the sensor drivers.
- Plots update in real time.
- Current sensor value displayed above each plot.

Global Controls

- Connect
- Disconnect
- Start Logging
- Stop Logging

Global Plot Controls

- Time Range
    - 5 min
    - 30 min
    - 2 hour
    - 8 hour
    - 48 hours

Per-Sensor Controls

- Auto Y-axis
- Manual Y-axis Minimum
- Manual Y-axis Maximum
